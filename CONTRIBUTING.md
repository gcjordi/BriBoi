# Contributing

Thank you for considering contributing to BriBoi.

## Ways to Contribute
- Code
- Documentation
- Biodiversity research
- Data science
- AI models
- Bug reports

## Pull Requests
1. Fork repository
2. Create feature branch
3. Commit changes
4. Open Pull Request

## Security
Never commit:
- API keys
- Tokens
- Passwords
- Personal data

All contributions are reviewed before merge.
