Fonts

La versió inicial consulta fonts obertes sense claus API obligatòries:

  • Open-Meteo Forecast API
  • Open-Meteo Air Quality API
  • ECDC RSS
  • WHO RSS
  • Copernicus Climate RSS
  • European Environment Agency RSS
  • ReliefWeb RSS
  • PubMed RSS configurat com a font opcional de senyals científics

Es poden afegir GBIF, Copernicus CDS, Europe PMC o fonts institucionals locals en futures versions.
